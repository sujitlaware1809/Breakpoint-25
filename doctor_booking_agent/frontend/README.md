# Doctor Booking Frontend - Next.js

Voice AI powered doctor appointment booking interface for Breakpoint Hackathon 2025.

## Quick Start

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)
