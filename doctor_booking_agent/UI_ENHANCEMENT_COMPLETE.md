### Design System

**Colors:**
- Primary: `#667eea` to `#764ba2` (Purple gradient)
- Success: `#10b981` to `#059669` (Green gradient)
- Error: `#ef4444` to `#dc2626` (Red gradient)
- Text: `#374151` (Dark gray)
- Hints: `#6b7280` (Medium gray)

**Typography:**
- Headlines: 32px, weight 800
- Subtext: 16px, weight 400
- Labels: 14px, weight 600
- Badges: 12px, uppercase, letter-spacing 0.5px

**Spacing:**
- Card padding: 48px
- Form gaps: 24px
- Input padding: 12px 16px
- Border radius: 10px (inputs), 16px (cards), 24px (main)

**Effects:**
- Backdrop blur on header and overlays
- Box shadows: 0 20px 60px for main card
- Hover transforms: translateY(-2px)
- Focus states: Border color + shadow ring
- Smooth transitions: 0.2s to 0.3s
